# AI_Connect_Four

## Rules
이 AI는 Search Algorithm을 기반으로 한 2종류의 Decision rule을 사용합니다.<br>
1. MiniMax Algorithm + Alpha-beta pruning
2. MCTS (Monte Carlo Tree Search)
